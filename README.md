\# RepuestoPro - SaaS para Tiendas de Autopartes



Sistema web completo para tiendas de repuestos automotrices en Perú.



\## Funcionalidades principales



\- \*\*Login multi-tenant\*\* (varias tiendas en un mismo sistema)

\- \*\*Dashboard\*\* con estadísticas

\- \*\*POS (Punto de Venta)\*\* completo con carrito

\- \*\*Facturación electrónica SUNAT\*\* (simulación Nubefact)

\- \*\*Búsqueda de productos\*\* por código y descripción

\- Diseño moderno y responsive



\## Tecnologías



\- Next.js 15 (App Router)

\- TypeScript

\- Tailwind CSS

\- Lucide React (iconos)
