# RepuestoPro - SaaS para Tiendas de Autopartes

Sistema completo tipo ERP + POS para tiendas de repuestos automotrices en Perú, con enfoque en **facturación electrónica SUNAT**.

## Funcionalidades actuales

- Login con selección de tienda (Multi-tenant)
- Dashboard con estadísticas
- POS completo con carrito de ventas
- Emisión simulada de comprobantes (Boleta/Factura)
- Facturación electrónica (simulación Nubefact)
- Diseño moderno y responsive

## Tecnologías usadas

- **Next.js 15** (App Router)
- TypeScript
- Tailwind CSS
- Lucide React

## Próximas funcionalidades

- Inventario con búsqueda por vehículo (marca, modelo, año)
- Gestión de clientes
- Reportes y gráficos
- Integración real con API de Nubefact
- Órdenes de compra a proveedores
